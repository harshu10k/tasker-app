## FIRST TIME SETUP

1. Open app
2. Click "Add New Task"
3. Fill in:
   - Title: What you need to do
   - Date: When
   - Time: What time
   - Priority: How urgent
   - Category: What type
4. Click "Add Task"
5. Your task appears!

=================================================================

## MOST IMPORTANT BUTTONS

Circle (left side) = Mark task complete
Trash icon (right side) = Delete task
Search box = Find tasks
Filter buttons = Show different tasks
Tag icon (bottom) = Manage categories

=================================================================

## NOTIFICATIONS

- First time: Browser asks "Allow notifications?"
- Click "Allow"
- Get reminder 5 minutes before task time
- Both desktop alert and app notification

=================================================================

## FILTERS EXPLAINED

PENDING - Shows tasks you haven't done
TODAY - Shows only today's tasks
HIGH PRIORITY - Shows only urgent tasks
COMPLETED - Shows finished tasks
ALL - Shows everything

=================================================================

## PRIORITY COLORS

Green = Low (not urgent)
Yellow = Medium (should do)
Red = High (do first)

=================================================================
